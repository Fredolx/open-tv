Changes this version

  • Added support for series without seasons
  • Added support for episodes without seasons
  • Fixed a bug where a different episode would play than the one selected
  • Fixed xtream bugs
  • Fixed a bug where the search bar would show 'undefined'

Changes since this major version:

  • Vastly improved navigation. Keyboard navigation should be a lot smoother. Use Escape or Backspace to go back in any view. Press F1 for more details. Please send feedback if you like keyboard-centric apps.
  • Categories view now follows your media type filters. Only want to see categories of vods? It's here.
  • Series are now organized in seasons. This was probably the most requested feature.
  • Playback of channels can be cancelled by just clicking on them again. No more zombies.
  • Squashed a ton of xtream bugs
  • Open TV now remembers your last search before entering a category, series or season
  • Re-added the scrollbar.
  • All episodes of a season are added to the queue when playing an episode.
  • Fixed a bug where you couldn't use left or right arrow keys in the search bar.
  • The linux flatpak package now uses mpv as base, which means it should be kept up-to-date with the latest dependencies in a more consistent fashion. If you had problems with the flatpak package, please try it now.
  • Updated all dependencies
